# 演算子の結合優先度

| 優先順位 | 演算子種類 | 演算子  | 結合性        |
|:---------|:-----------|:--------|:--------------|
| 3        | グループ化 | `( a )` | -             |
| 2        | 乗算 (*)   | `a * b` | `(a * b) * c` |
|          | 除算 (/)   | `a / b` | `(a / b) / c` |
| 1        | 加算 (+)   | `a + b` | `(a + b) + c` |
|          | 減算 (-)   | `a - b` | `(a - b) - c` |
